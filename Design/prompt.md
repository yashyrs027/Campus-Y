# Google Stitch UI Design Prompt

## Project Name

Smart College Event Management and Participation Platform (CampusConnect)

## Objective

Design a modern, responsive, and user-friendly web application for a Smart College Event Management and Participation Platform. The platform should simplify event creation, approval, registration, participation tracking, certificate generation, and administration for colleges.

## Design Style

* Modern SaaS dashboard
* Clean and minimal interface
* Professional and responsive layout
* Light theme with blue and white color palette
* Rounded cards and buttons
* Consistent spacing and typography
* Accessible UI with intuitive navigation
* Desktop-first with responsive support for tablet and mobile devices

## User Roles

* Student
* Club President / Vice President
* Faculty Coordinator
* Head of Department (HOD)
* Administrator
